# switch
Manage git worktrees.

**Usage:** `/switch` · `/switch <N>` · `/switch clean`

Invoke `tuti-workflow-master`:
> "Worktree management: $ARGUMENTS. No arg: list all worktrees with their issue numbers and git status (clean/dirty/ahead/behind). Issue number: find that issue's worktree and show the cd command. 'clean': run git worktree prune and show what was removed."
